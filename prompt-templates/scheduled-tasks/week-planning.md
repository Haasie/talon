# Week planning

Sunday evening overview of the upcoming week. Designed to review over dinner or with a partner.

<!-- Why: Starting Monday with a full picture of the week means fewer surprises.
     Identifying focus days and heavy days in advance lets you protect deep work
     time and set expectations. -->

<!-- Schedule: Sunday evening, e.g. "0 19 * * 0" -->

## Steps

1. **Calendar overview** -- Pull the full week from all configured calendars. Group by day.

2. **Key deadlines** -- Check your issue tracker for tickets with due dates this week. Check memory for any noted deadlines.

3. **Focus opportunities** -- Identify days with light meeting schedules where deep work is possible.

4. **Heavy days** -- Flag days with back-to-back meetings or early starts.

5. **Personal items** -- Surface anything from personal calendars or memory that affects the week (appointments, events, commitments).

6. **Open items carried forward** -- Check memory and recent conversations for things that need attention this week.

## Format

```
## Week ahead -- [Date Range]

### Monday
- [Work events]
- [Personal events]
- Focus window: [time range]

### Tuesday
...

### Highlights
- Best focus days: [days]
- Heavy days: [days]
- Deadlines: [items]
- Personal: [relevant items]
```
